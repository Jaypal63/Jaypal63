What is the difference between the 'id' attribute and the 'class' attribute of HTML elements? With Example.

Ans:-

ID:- 

         - Id attribute is used to specify a unique id for an HTML element.
         - Id name must only be used by one HTML element within the page.
         - You cannot have more than one element with the same id in an HTML document.
          Ex:-
          :<h1 id="title">jaypal</h1>

CLASS:- 

        - Class attribute is used when the same style must be applied to multiple HTML elements on the same web page. 
        - Class naming is case sensitive.
        - You can use multiple classes on the same element.
         Ex:-
         :<h1 class="title"> Question7 </h1>
         <span class="title"> Question7 </span>
