Que:- How can you convert the string of any base to an integer in JavaScript?

Ans:-

parseInt() function (or a method) is used to convert the passed-in string parameter or value to an integer value itself.

Use Integer.parseInt() to Convert a String to an Integer

Example:- 
            parseInt("5"); = 8
            parseInt(" 8 "); = 8
            parseInt("8", 2); = NaN
            parseInt("8.5"); = 8
            parseInt(" number 8 "); = NaN
