Que:-What are undeclared and undefined variables?

Ans:-

Undefined variable :-

  • An undefined variable in the source code of a computer program is a variable that is accessed in the code but has not been declared by that code.
  • It occurs when a variable has been declared but has not been assigned any value. Undefined is not a keyword.
exameple:- 
        let geek;
        undefined
        console.log(geek)

undeclared variables :-

  • It occurs when we try to access any variable that is not initialized or declared earlier using the var or const keyword
example:- 
        // ReferenceError: myVariable is not defined
console.log(myVariable)
