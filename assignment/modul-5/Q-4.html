(4)Canvas and SVG tags

Ans:-

1.SVG

  • The Scalable Vector Graphics is an XML based image format that is used to define two-dimensional vector-based graphics for the web.
  • SVG images can be created and edited with any text editor,such as notepad.
  • SVG has better scalability. So it can be printed with high quality at any resolution.
  • SVG can be modified through script and CSS.
  • SVG gives better performance with smaller number of objects or larger surface.

    2.Canvas

  • Used to draw graphics on the fly,via scripting(like JavaScript).
  • The element is only a container for graphics. You must use a script to actually draw the graphics.
  • Canvas has poor scalability. So it is not suitable for printing on higher resolution.
  • Canvas can be modified through script only.
  • Canvas gives better performance with smaller surface or larger number of objects.
