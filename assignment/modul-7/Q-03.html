Que:- How to create slider with animation?

Ans:-

If the elements have been slid up, slideToggle() will slide them down. $(selector).slideToggle(speed,callback);

The optional speed parameter can take the following values: "slow", "fast", milliseconds. The optional callback parameter is a function to be executed after the sliding completes.

jQuery slide Methods

With jQuery you can create a sliding effect on elements.

  • slideDown()
  • slideUp()
  • slideToggle()
