Que:- Write the code for adding new elements dynamically?

Ans:-

Code for adding new elements dynamically.

New elements can be created in JS by using the createElement() method.

Syntax:-

            document.createElement("tagName");  
                // Where tagName can be any HTML 
                // tagName like div, ul, button, etc.
                
                // newDiv element has been created
                For Eg: let newDiv = document.createElement("div");
