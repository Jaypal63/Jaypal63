How can you convert the string of any base to an integar in JavaScipt?

Ans:-

In JavaScript parseInt() function or method is used to convert the string parameter or value to an integer value itself.

This function returns an integer of the base which is specified in the second argument of the parseInt() function.

Example :-

            parseInt("5"); = 8
            parseInt(" 8 "); = 8
            parseInt("8", 2); = NaN
            parseInt("8.5"); = 8
            parseInt(" number 8 "); = NaN
