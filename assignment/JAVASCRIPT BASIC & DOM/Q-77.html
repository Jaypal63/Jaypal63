Q-77. What are all the looping structures in JavaScript?

Ans :

looping structure are allow repeatedly Execute The code if condition is true

- Looping structures in JavaScript

1. for loop

2. while loop

3. do while loop

4. for of loop

5. for in loop

Example :

<script>

        1. for loop:

        for (initialization; condition; update) {
            // code to be executed
          }
          

 
      
      2. while loop:

      while (condition) {
        // code to be executed
      }
      

     
 

      3. do while loop:
      
      do {
        // code to be executed
      } while (condition);
      

      
    
     

     4. for of loop:

     for (variable of iterable) {
        // code to be executed
      }
      


      

    5. for in loop:

    for (variable in object) {
        // code to be executed
      }
