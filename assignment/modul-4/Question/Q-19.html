Que:-What is the rule set?

ans:-

A rule set is aon type of statement ,which tells the browers how to show the HTML element in the webpages or web site

  1. A CSS rule consists of a selector and a declaration block.
  2. The selectors points to the HTML element you want to style.
  3. The declaration block contains one or more declarations separated by semicolons.
  4. Each declaration includes a CSS property name and a value separated by colon.
  5. Multiple CSS declarations are separated with semicolons and declaration blocks are surrounded by curly braces.
example:- 
        p {
            color: red;
            text-align: center;
          }
        p = HTMl elements you want to style
        color and text-align = property
        red and center = property value
