Q-4 What do you know about the Bootstrap Grid System?

Ans: The Bootstrap Grid System is a responsive grid system used in web development to create layouts that adapt to various screen sizes and devices.

The Bootstrap Grid System is based on a 12-column grid layout.

The grid system is inherently responsive, meaning that it automatically adjusts the layout and sizing of elements based on the screen size and device.

Bootstrap provides a set of CSS classes to define the layout of content within the grid.

Overall, the Bootstrap Grid System simplifies the process of creating responsive and flexible layouts, making it a popular choice for web developers.
