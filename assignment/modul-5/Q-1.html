Que:- What are the new tags added in HTML5?

Ans:-

Many new tags have been added in HTML5

  • content tags
  • Structural and Semantic Tags
content tags
  1. audio:- audio tag use to defines sound content within HTML document.
  2. video:- video tag use to defines video content within HTML document.
  3. embed:- it defines a container for external files/application/media.
  4. source:- iT defines multiple media resources for the media elements.
  5. track:- it defines text tracks for <audio> and <video> files
  6. datalist:- represent predefined list for input element.
  7. output:- It is used a container element to represent the output of a calculation or outcome of user action.
Structural and Semantic Tags
  1. header:- defines the introductory or navigational content of the webpage.
  2. footer:- represents the footer section of the webpage.
  3. main:- specifies the main content of the HTML document.
  4. mark:- represent the text which is highlighted or marked for reference or notation purposes.
  5. article:- defines the independent or self-contained content of a webpage.
  6. aside:- It defines the content which provide information about the main content.
  7. details:- defines additional information which only visible as per user demand.
  8. summary:- defines summary or caption for a element which can be clicked to change the state of element.
