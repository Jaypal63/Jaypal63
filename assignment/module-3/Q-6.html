Question-6:- What are the 'class' attribute in HTML? With Example.

Ans:- 
        -The class attribute is mostly used to point to a class in a style sheet.
        - With the specified class name you can work it through CSS and point in a stylesheet.
        - The class attribute specifies one or more classnames for an element.
        -We can use the same class attribute name whith different element.

        EX-
           <h2 class="title">hello world </h2>
           <titel class="title"> hello world </titel>
