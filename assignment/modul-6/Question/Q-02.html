Que:- What is the use of isNaN function?

Ans:-
  • JavaScript NaN stands for not-a-number
  • isNaN() The isNaN() function determines whether a value is NaN , first converting the value to a number if necessary. Because coercion inside the isNaN() function can be surprising, you may prefer to use Number.isNaN() .
  • isNaN() returns true if a number is Not-a-Number
  • isNaN() converts the value to a number before testing it.
        < script >
                console.log(isNaN(5)):
                console.log(isNaN("hi frends")):
            < / script >
