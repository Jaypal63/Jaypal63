Que:- What is === operator?

ans:-

(===) operator is one type of comparison operator which is called "strict operator"

The strict equality ( === ) operator checks whether its two operands are equal, returning a Boolean result

Unlike the equality operator, the strict equality operator always considers operands of different types to be different.

this operator performs type casting for equality

Example:- 
            
  • 7 === 7 (true)
    • 
            
  • '7' === 7 (False)
