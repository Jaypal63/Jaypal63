Que:- How to embed audio and video in a webpage?

  • <Audio> <video> tags can be used to embed audio video on webpages.
  • audio and video tags have many attributes
audio-video tags attributes
  1. Autoplay:- this attribute if specified the video will automaticlly begin to play back as soon as it can do so without stopping to finish loading the data
  2. controls:- attribute is present it will allow the user to control video playbck
  3. loop:- this attribute if specified will allow video automatically seek back to the start after reaching at the end
  4. src:- URL of the audio and video file
Audio

    2.             <audio controls >
